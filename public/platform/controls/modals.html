Cannot Find Wallet

To use the Yeek Exchange, your browser must be connected to the Blockchain. Please do one of the following:

  • (Desktop Only) Install a compatible wallet as a browser plugin. We recommend Metamask
  • View this page in a browser that has a built in wallet. We recommend Trust on iOS or Toshi on Android

Once you install the appropriate software, please reload the page. Because this is a fully decentralized exchange (no server!), none of the features will work without Blockchain connectivity.

If you require support, please post on our Telegram group and an engineer will assist

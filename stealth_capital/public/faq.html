What Software Do I Need? Because this is a decentralized exchange, trades are executed directly on the Ethereum Blockchain. This website provides a user interface to help you interact with Smart Contracts on the Blockchain but but the actual transaction is initiated from your Ethereum wallet.

There are two ways to bridge the gap between your browser and the Blockchain:

1. Use one of the new browsers that contains a built in wallet: Trust or Toshi are recommended for iOS and Android mobile devices; Mist is for desktop users on Windows, Mac, or Linux.

2. Download a wallet plugin for your existing browser. Metamask works with both Chrome and Safari, for desktop users only (if on mobile, use one of the browsers recommended above) Where Are My Tokens?
